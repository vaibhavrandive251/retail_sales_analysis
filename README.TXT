Project Title: Retail Sales Data Analysis using Python

Objective:
The objective of this project is to analyze retail sales data and identify
revenue trends, top-performing products, and monthly sales performance.

Tools & Technologies Used:
- Python
- Pandas
- Matplotlib

Project Description:
In this project, I performed data cleaning and preprocessing on a retail sales dataset.
I converted date columns into proper datetime format and created new features such as Month and Year.
I calculated total revenue generated and identified top 5 revenue-generating products.
I also analyzed monthly sales trends using data visualization.

Key Insights:
- Total revenue calculated from dataset
- Identified top 5 best-selling products
- Visualized monthly sales trends
- Performed exploratory data analysis (EDA)

Conclusion:
This project demonstrates my ability to clean, analyze, and visualize structured data
using Python for business insights.
